## Command list

1. **EXIT**<br>
   `exit` 프로그램 종료<br>
2. **SELECT**<br>
   `select [n]` n번 에어컨 선택<br>
   `select all` 모든 에어컨 선택<br>
3. **POWER**<br>
   `power on` 전원 켜기<br>
   `power off` 전원 끄기<br>
4. **TEMPERATURE**<br>
   `temp up` 온도 1도 증가<br>
   `temp down` 온도 1도 감소<br>
   `temp [n]` 온도 n도로 설정<br>
5. **MODE**<br>
   `mode next` 다음 모드로 설정<br>
   `mode [cool/dehum/warm/blow]` 특정 모드로 설정<br>
6. **FAN**<br>
   `fan up` 풍량 증가<br>
   `fan down` 풍량 감소<br>
   `fan [low/mid/high/power]` 특정 풍량으로 설정<br>
7. **CALM**<br>
   `calm on` 무풍 켜기<br>
   `calm off` 무풍 끄기<br>
