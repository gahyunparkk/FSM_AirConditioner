## Command list

1. EXIT
   `exit` 프로그램 종료
2. SELECT
   `select [n]` n번 에어컨 선택
   `select all` 모든 에어컨 선택
3. POWER
   `power on` 전원 켜기
   `power off` 전원 끄기
4. TEMPERATURE
   `temp up` 온도 1도 증가
   `temp down` 온도 1도 감소
   `temp [n]` 온도 n도로 설정
5. MODE
   `mode next` 다음 모드로 설정
   `mode [cool/dehum/warm/blow]` 특정 모드로 설정
6. FAN
   `fan up` 풍량 증가
   `fan down` 풍량 감소
   `fan [low/mid/high/power]` 특정 풍량으로 설정
7. CALM
   `calm on` 무풍 켜기
   `calm off` 무풍 끄기
